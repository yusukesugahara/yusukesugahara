## 🚀 Featured Portfolio

### ReviewFlow

[GitHub Repository](https://github.com/yusukesugahara/ReviewFlow)

**ReviewFlow** は、市役所・自治体などの申請業務を想定した、申請・承認ワークフロー管理アプリです。

補助金申請、施設利用申請、各種届出などの業務を想定し、申請フォームの作成から、申請受付、承認、差し戻し、再提出、CSV出力、監査ログ確認までを一貫して管理できるアプリとして開発しています。

このプロジェクトでは、単なるフォーム送信ではなく、申請後の業務フローまで含めて管理できることを重視しています。

例えば、以下のような観点を設計に取り入れています。

- 誰が申請内容を確認・承認するのか
- 申請が現在どの状態にあるのか
- 差し戻し時に、どの項目を修正してほしいのか
- 差し戻し後に再提出できるか
- 誰がいつ、どの操作を行ったのか
- 後から監査ログとして確認できるか

特徴は、**ログインアカウントを持たない外部ユーザーからの申請受付を想定している点** です。

職員や管理者向けの業務管理だけでなく、住民や事業者などの申請者側の利用ハードルを下げることも意識しています。

- 承認フローの設定
