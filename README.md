## 🚀 Featured Portfolio

### ReviewFlow

[GitHub Repository](https://github.com/yusukesugahara/ReviewFlow)

**ReviewFlow** は、市役所・自治体などで利用される申請業務を想定した、申請・承認ワークフロー管理アプリです。

補助金申請、施設利用申請、各種届出などの業務を想定し、申請フォームの作成から、申請受付、承認、差し戻し、再提出、CSV出力、監査ログ確認までを一貫して管理できるアプリとして開発しています。

特徴は、**ログインアカウントを持たない外部ユーザーでも申請できる設計** です。

職員や管理者向けの業務管理だけでなく、住民や事業者などの申請者側の利用ハードルを下げることを意識しています。

このプロジェクトでは、単なるフォーム送信ではなく、実際の業務で必要になる以下のような観点を重視しています。

* 誰が申請内容を確認・承認するのか
* 差し戻し時に、どの項目を修正してほしいのか
* 申請が現在どの状態にあるのか
* 誰がいつ、どの操作を行ったのか
* 後から監査ログとして確認できるか

業務システムとして運用することを想定し、申請後の確認・承認・差し戻し・再提出までの流れを管理できるように設計しています。
